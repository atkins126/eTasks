Sobre o eTasks

O eTasks é um aplicativo Android e Windows desenvolvido em Delphi que utiliza os recursos do Firebase como backend. Agora o objetivo é trazer uma segunda versão melhorada usando Delphi, Skia4Delphi para melhorar os gráficos, eFirebase para manipular funções do Firebase e, também, uma versão web que será desenvolvida com o framework Blazor WebAssembly e usufruindo da linguagem C#. 

O objetivo do aplicativo eTasks é gerenciar tarefas de forma extremamente simples e rápida, trazendo um bom humor para um aplicativo singelo. Para torná-lo ainda mais útil, o eTasks fornece a possibilidade de gerenciar listas de mercado e manter metas em foco.
